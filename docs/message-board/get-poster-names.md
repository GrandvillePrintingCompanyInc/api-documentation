# Get Poster Names

Returns an array of `postedBy` names pulled from previous messages.

```plaintext
GET /api/MessageBoard/GetPosterNames
```

### Parameters

| Parameter     | Required | Type                                   | Description                                        |
| ------------- | -------- | -------------------------------------- | -------------------------------------------------- |
| key           | yes      | `string`                               | API key provided by Grandville Printing Company    |

### Reponse Body

```typescript
interface {
    success: boolean,
    error: string,
    data: string[],
}
```
